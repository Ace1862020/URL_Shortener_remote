# 短網址產生器-實作-URL_Shortener
![](https://github.com/Ace1862020/URL_Shortener_remote/blob/master/public/url-shortener-index.jpg)

## Features - 產品特色
* 輸入網址，會回傳格式化的短網址
* 使用短網址可以連向原網站
* 如果未輸入任何內容就shorten，會顯示錯誤
* 短網址為5碼英數組合(含大小寫)
* 點擊Copy可以複製短網址在您的剪貼簿裡
* 點擊Back按鈕or大icon，網頁會回到初始狀態
